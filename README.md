# Quizzes Demo App

A small demo app for the [Rrogramming Ruby](http://fmi.ruby.bg) course.

## Install

To install the app, clone the repo, go into the cloned dir and run:

```shell
bin/setup
```

## Run the app

Run the app with:

```shell
bin/rails server
```

And open it at [http://localhost:3000](http://localhost:3000).
